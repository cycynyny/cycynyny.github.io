MATLAB programming, numerical problem solving, and engineering computation for mechanical engineers. Students develop practical skills in scientific computing through hands-on projects connected to real thermal-fluid engineering problems.

## Topics Covered

- MATLAB fundamentals: data types, variables, and operators
- Control flow: conditionals, loops, and functions
- Plotting and data visualization
- File I/O and data processing
- Applied numerical methods: root finding, numerical integration, curve fitting
- Engineering problem solving with computation

## Course Details

- **Units:** 3 (1 lecture + 2 lab sections)
- **Offered:** Every semester since Fall 2024
- **Prerequisites:** None
